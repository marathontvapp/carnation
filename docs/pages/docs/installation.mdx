# Installation

Get started with Carnation by installing `carnation-ds`:

<Tabs items={["npm", "yarn"]} defaultIndex={1}>
  {/* prettier-ignore */}
  <Tab>
    ```sh copy
    npm i carnation-ds@alpha
    ```
  </Tab>
  {/* prettier-ignore */}
  <Tab>
    ```sh copy
    yarn add carnation-ds@alpha
    ```
  </Tab>
</Tabs>

## Motion

If you're using the motion primitives, be sure to install the peer dependency appropriate for your platform.

### Web

To use motion on the web, install the latest version of [Framer Motion](https://www.framer.com/motion/):

<Tabs items={["npm", "yarn"]} defaultIndex={1}>
  {/* prettier-ignore */}
  <Tab>
    ```sh copy
    npm i framer-motion@latest
    ```
  </Tab>
  {/* prettier-ignore */}
  <Tab>
    ```sh copy
    yarn add framer-motion@latest
    ```
  </Tab>
</Tabs>

### Native

To use motion on native, install the latest version of [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/):

<Tabs items={["npm", "yarn"]} defaultIndex={1}>
  {/* prettier-ignore */}
  <Tab>
    ```sh copy
    npm i react-native-reanimated@latest
    ```
  </Tab>
  {/* prettier-ignore */}
  <Tab>
    ```sh copy
    yarn add react-native-reanimated@latest
    ```
  </Tab>
</Tabs>
